Protocol Documentation

Welcome to HyperFlow, a next-generation real-time token streaming protocol built natively on the high-speed Vara Network. HyperFlow enables frictionless, continuous payment distribution per block, designed to support modern teams, DAOs, and digital subscriptions.

Why HyperFlow?

Traditional payroll and payout methods batch operations into massive weekly or monthly transfers. HyperFlow replaces these with continuous streams. Instead of waiting, the recipient's wallet balance increases dynamically every second. Our architecture is trustless, non-custodial, and engineered using Vara’s advanced actor model messaging system.

Quick Start Guide

Get up and running with HyperFlow on the Vara Testnet in four simple steps.

01
Connect SubWallet
Open the dashboard and connect your Polkadot-compatible extension like SubWallet, Polkadot.js, or Talisman.
02
Acquire VARA Gas
Use the sidebar link to go to the official Vara Faucet and request testnet VARA tokens to cover messaging transactions.
03
Mint GROW Tokens
Head to the GROW Token tab in the dashboard and mint 1,000 GROW from the testnet faucet to fund your vaults.
04
Start a Stream
Approve the TokenVault, deposit your GROW, and define your recipient's address to initialize your first live stream.

Architecture Overview

HyperFlow operates through a three-tier modular architecture designed for maximum performance, security, and composability.

Module Name Role Security Level
GROW ERC20 Contract Handles the fundamental asset ledger and permissions. Audited & Immutable
TokenVault Contract Stores users' deposited streaming collateral in locked escrows. Non-custodial, multisig-ready
StreamCore Contract Orchestrates the active streams, computes flow rates, and manages withdrawals. Dynamic calculation, Zero-custody

By splitting the vault and core execution layers, users can grant streaming permissions safely without placing their entire balance at risk, creating a premium standard for secure decentralized banking.

SDK Integration Guide

Integrate streams natively into your frontends using the HyperFlow JavaScript API. Leverage the lightweight SDK wraps directly from `window.API` in your dashboard scripts.

JavaScript Integration Example 
// Initialize HyperFlow API interface
const myAddress = "0xABCD...";

// 1. Fetch GROW token balance
try {
  const balanceInfo = await API.token.balance(myAddress);
  console.log("GROW Balance:", API.formatGROW(balanceInfo.balance));
} catch (error) {
  console.error("Failed to retrieve balance:", error);
}

// 2. Query active streams for sender
const streams = await API.streams.bySender(myAddress);
console.log("Total Outgoing Streams:", streams.length);
